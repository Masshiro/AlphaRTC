# AlphaRTC Modified Version

## Usage:
- Build the whole project using script:
```shell
. build_serverless.sh
```

- For the receiver client, using script:
```shell
. serverless_receiver.sh
```

- For the sender client, using script:
```shell
. serverless_sender.sh
```
